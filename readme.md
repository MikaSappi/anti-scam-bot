# How to use
1. Set your own website's URL in the `targetURL` field
2. Start the service
3. The script mimics real user load and is great for stress testing.

# How it works
It reads a list of 200 English first names and 200 English last names. It picks two random names from the list, combines them, and forms an email address for them with a list of email service providers. It sends more leads during the day time and less during the night. It also cycles through user agents to make it look like a proper contact instead of a bot.

## Disclaimer
It's a stupid idea to use this on legitimate websites you don't own. Don't do it.

## Why this exists
To fill online scammers' lead lists with fictional data. This script generates around a thousand fake leads in a day and will result in the scammers wasting their time on sending email to fictional people.
