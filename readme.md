# How to use
1. Go to your spam folder
2. Find an email that opens a contact form
3. Use the url in `targetURL` field
4. Make it a service and visit it after a few days. Usually these sites get down within the first 48 hours.

# How it works
It reads a list of 200 English first names and 200 English last names. It picks two random names from the list, combines them, and forms an email address for them with a list of email service providers. It sends more leads during the day time and less during the night. It also cycles through user agents to make it look like a proper contact instead of a bot.

Success is dependent on IP: run from a datacenter and you'll probably get blocked.

## Disclaimer
It's a stupid idea to use this on legitimate websites. Don't do it.

## Why this exists
To corrupt online scammers' lead lists. This script generates around a thousand fake leads in a day and will result in the scammers wasting their time on sending email to fictional people.
